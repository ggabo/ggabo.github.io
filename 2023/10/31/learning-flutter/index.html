Learning Flutter

Tue Oct 31 2023
  • Programming
  • web
  • desktop
  • mobile

So I recently got interested in flutter. As someone that has been a developer for some time, I wanted to be more deliberate about how I go about it, mostly answering these questions:

  • Why?
  • How?
  • Where?

# Why

This is simple. I tend to dislike doing frontend stuff, but I like graphics, image processing and stuff like that. I also want something that's multiplatform. There's electron and reactive native and PWAs and all that jazz, but I wanted something with batteries included as much as possible because of the "where" section. And flutter presented itself when I read about the localsend app[0] and I said "Huh? Cool". That's about the gist of it.

# How

I've known roadmap.sh[1] for a while now and I really wanted to use it as a guide for something (might use it for prompt engineering and data science). I'll also use other sources like the official site[2] and youtube channel[3] , but I'll check those boxes and see if it feels like my confidence as a Dart/flutter programmer increases acordingly. The sintax is c#-like enough so I'm not annoyed at it (I don't love any particular sintax, if I can get used to it, I'm fine).

# where

Where as in the domain of projects I'm going to apply it. Mostly small apps with some image processing purpose (my image editor app is in dire need of some attention) that I might want to put in multiple places, like my website and also my google play developer account that has been gathering dust for years.

# Conclusion

So this is the basic of it. It might crash and burn horribly or I might love it. We'll just have to wait and see.
